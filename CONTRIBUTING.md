## How to Contribute (Step-by-Step)

### 1. Fork the Repository
Click **Fork** in the top-right corner of the repo.

### 2. Create a New Markdown File
Choose the most relevant category and create a new file:

```text
use-cases/<category>/<your-descriptive-slug>.md
```
Example:

```
use-cases/engineering/debugging-backend-incidents-with-ai.md
```
